# The Awakened Minds Collective

A bespoke website built with:

- React (TypeScript)
- Vite
- Tailwind (v4)
- Netlify

## Setup

First, install the node modules:

```bash
npm i
```

Then, run the project using **one** of:

```bash
npm run dev
```

```bash
npm run localhost
```

```bash
npm run network
```
